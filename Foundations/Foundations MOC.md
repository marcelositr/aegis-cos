# Foundations MOC

## Core Concepts

- [[Abstraction]] — Managing complexity through layered simplification
- [[Algorithms]] — Step-by-step procedures for computation
- [[Complexity]] — Big-O analysis, time and space complexity
- [[Computability]] — What can and cannot be computed
- [[DataStructures]] — Organizing data for efficient access and modification
- [[InformationTheory]] — Quantification, storage, and transmission of information
- [[SystemsThinking]] — Holistic analysis of complex systems and their interconnections

## Reasoning Path

1. Start with [[Abstraction]] — understand how to manage complexity
2. Learn [[DataStructures]] + [[Algorithms]] — the building blocks
3. Analyze with [[Complexity]] — measure efficiency
4. Understand limits with [[Computability]] — know what's impossible
5. Apply [[InformationTheory]] — understand data fundamentals
6. Think holistically with [[SystemsThinking]] — connect everything
