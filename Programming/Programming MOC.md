# Programming MOC

## Languages

- [[TypeScript]] — Typed superset of JavaScript with compile-time type checking
- [[JavaScript]] — Multi-paradigm language with event-driven, functional, and prototype-based features
- [[Kotlin]] — Modern JVM language with null safety and coroutines
- [[Swift]] — Apple ecosystem language with safety and performance focus

## Concurrency

- [[Concurrency]] — Handling multiple tasks simultaneously with threads, async/await, actors, and processes

## Reasoning Path

1. Choose language based on domain: [[TypeScript]]/[[JavaScript]] for web, [[Kotlin]] for Android/backend, [[Swift]] for iOS
2. Master concurrency: [[Concurrency]] → understand threads, async, actors, processes
3. Apply language-specific patterns and best practices
4. Connect to testing, architecture, and performance
